Formalisation and Analysis of Component Dependencies

Maria Spichkova

28 April 2014

Abstract

This set of theories presents a formalisation in Isabelle/HOL of data dependencies between components. The approach allows to analyse system structure oriented towards efficient checking of system: it aims at elaborating for a concrete system, which parts of the system are necessary to check a given property.
